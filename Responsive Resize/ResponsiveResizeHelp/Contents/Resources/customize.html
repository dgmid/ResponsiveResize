The Customize Menu

The Customize Menu gives you access to various options that let you change the way the application works.

Add/Remove Devices…

Opens the Preferences Panel where you can choose or create new devices to add to the Devices Bar

Switch to Light / Dark Theme

Toggle between the Light or Dark theme.

Application Preferences…

Opens the Preferences Panel where you can change the following settings:

  1. Adjust the display of the Recent Sites Menu go to page…
  2. Choose to open the last visited site on application launch go to page…
  3. Adjust the step amount of the + /- increment buttons go to page…
  4. Show / Hide the Manufacturer's name on the Device buttons go to page…
  5. Show the User Agent String summary in the Status Bar go to page…

Reset Application…

Restore the application to its factory settings.
