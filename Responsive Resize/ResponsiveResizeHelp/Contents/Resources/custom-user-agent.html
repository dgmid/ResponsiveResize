Responsive Resize

Using a custom User Agent String

Responsive Resize lets you enter a custom user agent string

  1. Open the 'User Agent' menu and select 'Custom User Agent String…' or use the keyboard shortcut ⇧⌘C
  2. When the Custom User Agent String dialog opens, paste in your string and click 'Apply Custom User Agent String' or press return.
  3. As with the preset User Agent Strings, the User Agent info will be displayed in the status bar.
