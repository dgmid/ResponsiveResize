The URL Bar

The URL Bar is where you navigate to the site you wish to test. It also contains the 'Recent Sites' menu.

Using the URL bar

  1. Type or paste the URL into the input field on the URL bar.
  2. Hit return or click on the load icon

Using the Recent Sites Menu

  1. Click the icon to open the Recent Sites Menu
  2. The menu displays a list of any recently visited sites. You can clear the list by selecting 'Clear All Recent Sites'.

NOTE: you can select how may site are listed in the 'Recent Sites' in the preferences panel.
