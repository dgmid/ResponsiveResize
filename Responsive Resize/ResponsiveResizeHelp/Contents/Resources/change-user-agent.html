Responsive Resize

Changing the User Agent String

Responsive Resize allows you to easily change the User Agent String to anything you require.

  1. Open the 'User Agent' menu on the main menu bar.
  2. Select a new User Agent String from the menu.
  3. The active User Agent String will be persistent on close and relaunch of the application.
  4. If you have the preference set to display the User Agent String in the status bar, you will see a summary of the string in the status bar - hovering over it will reveal the full string.

NOTE: Responsive Resize also allows you to enter a custom User Agent String
